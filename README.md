# Gender Bias in Wikidata: Berufsklischees

## Projektbeschreibung
Dieses Projekt untersucht, inwiefern Wikidata geschlechtsspezifische Klischees in der Darstellung von Berufen widerspiegelt.

## Forschungsfrage
Inwiefern führt die Struktur von Wikidata zu einer Unterrepräsentation bestimmter Berufe und zu geschlechtsspezifischen Verzerrungen?

## Datenquelle
- Wikidata
- SPARQL-Abfragen

## Methodik
- Abfrage von Berufen (z. B. CEO, Büroberufe)
- Analyse der Geschlechterverteilung
- Untersuchung von Datenlücken (unterrepräsentierte Berufe)

## Ziel
Aufzeigen von Gender Bias und struktureller Unsichtbarkeit in Daten.
